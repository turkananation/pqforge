# pqforge Milestones

## M1 Architecture

Move from one service file to layered ownership and public exports.

## M2 Envelopes

Ship binary v1 and JSON/base64 v1 for encrypted and signed payloads.

## M3 Key custody

Support key import/export, passphrase wrapping, and app-provided key stores.

## M4 Cookbook APIs

Expose direct helpers for the major `pqcrypto` cookbook building blocks.

## M5 Developer experience

Keep first-screen APIs simple: generate, sign, verify, encrypt, decrypt.

## M6 Delivery

Keep CI, dry-run publish, docs, examples, and tests green before release.
